Advanced Valuables

What does this mod do?

You must be bored with Diamonds, Emerald, Iron, Gold, Netherite etc.
You use Diamonds and Irons for Protection, Emeralds for trading, Gold for
be protected from Piglins and Emeralds for trading.
No worries at all because this mod adds different tools, armors, valuable gems
and MORE!

Required Mod Loaders

Using NeoForge

NeoForge is required for NeoForge mods

Required Loader Version → 21.0.157 or above

Using Forge

Forge is required for Forge mods

Required Loader Version → 51.0.33 or above

Using Fabric

Fabric is required for Fabric mods

Fabric API Version → 0.104.0+1.21.1 or above

Required Loader Version → 0.16.5 or above

Dependencies

For Fabric

Mod Menu is an optional dependency
